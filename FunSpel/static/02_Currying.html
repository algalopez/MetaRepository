CURRYING

Technique of converting a function that takes multiple arguments into a sequence of functions that each take a single argument.

  • Manual application
  • Automatic Partial application

MANUAL

Example: Curryfied function

Not Curryfied: 
                    function sumAndMultiply(a, b, c) {
                        return (a+b)*c;
                    }
Curryfied: 
                    function curriedSumAndMultiply(a) {
                        return function(b) {
                            return function(c) {
                                return (a+b)*c
                            }
                        };
                    }
Results: 
                    >> sumAndMultiply(2,3,5)
                    <- 25
                    >> curriedSumAndMultiply(2)(3)(5))
                    <- 25

AUTOMATIC PARTIAL APPLICATION

Automatic partial application automatically curry the funcions

Example: Curryfied function

Haskell sample: 
                    add :: Int -> Int -> Int
                    add x y = x + y
                    
                    multiply :: Int -> Int -> Int
                    multiply x y = x * y
                    
                    sumAndMultiply:: Int -> Int -> Int -> Int
                    sumAndMultiply a b c = multiply (add a b) c
Demo: 
                    *Main> let multiplyBy11 = sumAndMultiply 7 4
                    *Main> :info multiplyBy11 
                    multiplyBy11 :: Int -> Int
                    *Main> multiplyBy11 3
                    33
