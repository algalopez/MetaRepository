COMPOSITION

Function signature: 
            (.)                     :: (b->c) -> (a->b) -> (a->c) 

            f . g                   = \ x -> f (g x)

Example: Composition of f(f(g(x))) where

  • f(x) = x+1
  • g(x) = x*10

                    composeFunctions:: Integer -> Integer
                    composeFunctions = \x -> ((+1).(+1).(*10)) x
